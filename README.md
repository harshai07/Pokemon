# Project Overview

This is a React application that displays a list of Pokémon fetched from the PokeAPI. It features a clean, responsive design with theme switching capabilities and error handling.

# API Used

- **PokeAPI** (https://pokeapi.co/api/v2/pokemon): Provides comprehensive Pokémon data including names, images, types, and other details used in the application.

# Features

- Fetch and display a list of Pokémon with pagination support
- Display detailed Pokémon information: name, image, and types
- Dark/Light theme toggle for user preference
- Error boundary component to handle and display errors gracefully
- Loading states with skeleton screens for better UX
- Responsive design that works on desktop and mobile devices

# Setup Instructions

1. Clone the repository to your local machine
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application
